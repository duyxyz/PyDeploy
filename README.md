# Python to EXE Converter

**Simple • Beautiful • Fully Vietnamese GUI • One-click .py → .exe**

Một click chuyển Python thành file EXE – cực nhanh, cực đẹp, hoàn toàn tiếng Việt!

### Features • Tính năng
- One File / Windowed mode (No Console)  
- Auto detect & add hidden imports for popular GUI frameworks  
  → Tkinter • CustomTkinter • PyQt5/6 • PySide2/6 • Kivy • Pygame  
- Custom icon (.ico) support  
- Drag & drop .py file  
- Real-time progress bar + detailed log  
- Output directly to `dist` folder next to your script  
- Clean, modern, resizable interface  

### Requirements • Yêu cầu
- Python 3.6+
- PyInstaller → `pip install pyinstaller`
